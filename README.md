# CareerHub Backend Microservices

This repository contains microservices for the CareerHub platform, converted from the monolithic backend.

## Microservices

1. **Job Service** – Handles job postings and CRUD operations.
2. **Company Service** – Handles company profiles.
3. **Review Service** – Handles user reviews.

## Tech Stack

- Java, Spring Boot
- REST APIs
- MySQL/PostgreSQL
- Maven
